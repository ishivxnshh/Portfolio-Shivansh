
# Shivansh Portfolio - Coming Soon

This is a minimal React project for Shivansh's portfolio. The site currently displays a visually appealing, centered, and responsive "Coming Soon" page.

## How to Run

1. Install dependencies:
	```
npm install
	```
2. Start the development server:
	```
npm run dev
	```

The site will be available at `http://localhost:5173` by default.

---

Replace the content with your portfolio when ready!
